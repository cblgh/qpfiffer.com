qpfiffer.com
============

This is the Hakyll-powered code for the qpfiffer.com blog.

Installation
------------

Requirements: cabal > 1.18.0, Hakyll

```
sudo apt-get instal alex happy
cabal sandbox init
cabal install hakyll --constraint='case-insensitive==1.0.0.1'
cabal install
qpfiffer.com build
```
